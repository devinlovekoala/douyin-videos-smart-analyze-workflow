# 视频文件目录

将屏幕录制的抖音短视频文件放置在此目录下。

## 支持的视频格式

- **WebM** (推荐，系统录屏默认格式)
- **MP4**
- **AVI**
- **MOV**

## 视频要求

- **来源**: 抖音屏幕录制
- **命名**: 任意名称（建议使用有意义的名称）
- **数量**: 建议 5-20 个视频

## 示例

```
download_videos/
├── dance_video_01.webm
├── food_cooking_02.webm
├── funny_cat_03.mp4
├── makeup_tutorial_04.webm
└── tech_review_05.webm
```

## 注意事项

- 确保视频文件完整可播放
- 视频时长建议在 15 秒到 3 分钟之间
- 视频分辨率建议 720p 或以上
- WebM 格式完全支持（系统录屏的原生格式）
